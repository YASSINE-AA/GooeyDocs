Detailed Description

Provides logging and debugging utilities with configurable log levels.

This header file defines an interface for logging messages, printing stack traces, and dumping memory contents for debugging purposes.

Saves the logged messages to a file.

This function writes all the accumulated log messages to a specified file. Each log message is written to a new line in the file. If the file cannot be opened, an error is printed to the standard output.

Parameters
pathThe path to the file where the log messages will be saved. If the file does not exist, it will be created. If it already exists, its contents will be overwritten.
